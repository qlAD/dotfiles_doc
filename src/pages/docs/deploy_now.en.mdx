### 1.3. Use sxhkd to set shortcut keys

[GitHub - baskerville/sxhkd: Simple X hotkey daemon](https://github.com/baskerville/sxhkd)

sxhkd is an X daemon that reacts to input events by executing commands.

Its configuration file is a series of bindings that define the association between input events and commands

The format of the configuration file supports a simple notation that maps multiple shortcuts to multiple commands in parallel.
