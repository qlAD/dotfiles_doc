# Do not pretend to be the site owner?

如果你在第三步遇到这个提示…

![Do not pretend to be the site owner](../_images/faqs/dont-pretend-to-be-site-owner.png)

这是因为我们在确保 **你** 是 **你本人**。如果你遇到这个提示，**请检查 `site.config.js` 里的 `userPrincipalName` 是否填错了**。

你填写的账户必须与你在第二步中登录的微软账户 **完全一致** ！😡
