# 托管文件与图片

是，它可以做图床（或者文件网盘），你可以将它生成的文件直链嵌入到任何其他网站上。

下面的 GIF 就是个例子：

![nyan cat](https://drive.swo.moe/api?path=/%F0%9F%A5%9F%20Some%20test%20files/nyancat.gif&raw=true)

你可以在文件列表或预览页面复制直链，或者按照下面的格式生成：

```markdown
![nyan cat](https://drive.swo.moe/api?path=/%F0%9F%A5%9F%20Some%20test%20files/nyancat.gif&raw=true)
```
