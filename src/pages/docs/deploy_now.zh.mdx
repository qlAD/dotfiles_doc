## 1、安装 Arch Linux

### 1.1、参考

### 1.2、使用 Bspwm 作为 窗口管理器

[GitHub - baskerville/bspwm: A tiling window manager based on binary space partitioning](https://github.com/baskerville/bspwm)

• 它_只是_一个窗口管理器（WM）。
• 它由几个易于配置的脚本管理。
• 它默认支持窗口之间的间隙。

### 1.3、使用 sxhkd 进行快捷键设置

[GitHub - baskerville/sxhkd: Simple X hotkey daemon](https://github.com/baskerville/sxhkd)

sxhkd 是一个 X 守护进程，它通过执行命令对输入事件做出反应。

它的配置文件是一系列绑定，定义了输入事件和命令之间的关联

配置文件的格式支持将多个快捷方式并行映射到多个命令的简单表示法。

### 1.4、使用 Ly 作为登录管理器

[GitHub - fairyglade/ly: display manager with console UI](https://github.com/fairyglade/ly)

Ly - TUI 显示管理器
